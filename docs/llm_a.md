## Cursos a desarrollar

1. [AI Fluency: Framework & Foundations de Anthropic](https://anthropic.skilljar.com/ai-fluency-framework-foundations)

2. [LLM Course de Hugging Face](https://huggingface.co/learn/llm-course/): Un curso largo y bastante completo sobre los básico de LLM aunque muy basado en el uso de las librerías de Hugging Face.

3. [AI Agents Course de Hugging Face](https://huggingface.co/learn/agents-course): Un curso muy completo que revisa 4 de las plataformas más importantes para desarrollar agentes de IA con LLMs.

4. [Model Context Protocol (MCP) Course de Hugging Face](https://huggingface.co/learn/mcp-course): Un curso corto que explica el protocolo MCP para extender las capacidades de los LLMs.


## Recursos sobre modelos grandes de lenguaje (LLM)

1. [The Illustrated Transformer](http://jalammar.github.io/illustrated-transformer/): Una explicación visual del modelo Transformer, que es la base de muchos LLMs.
2. [Attention is All You Need](https://arxiv.org/abs/1706.03762): El artículo original que introdujo el modelo Transformer.
3. [Hugging face](https://huggingface.co/): Una plataforma popular para trabajar con modelos de lenguaje preentrenados.
4. [Anthropic Academy](https://www.anthropic.com/learn): Muy centrado en Claude, pero trae ideas válidas para todos los LLMs.
